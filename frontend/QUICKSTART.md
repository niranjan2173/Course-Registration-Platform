# Quick Start Guide

## ⚡ Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
cd course-registration-app
npm install
```

### Step 2: Start the App
```bash
npm start
```

### Step 3: Run on Device
- **Android**: Press `a` in terminal or scan QR code with Expo Go app
- **iOS**: Press `i` in terminal or scan QR code with Expo Go app
- **Web**: Press `w` in terminal

## 🔑 Login
- **Student**: `student@college.edu` / `student123`
- **Admin**: `admin@college.edu` / `admin123`

## 📱 If You Don't Have Expo Go
1. Download from App Store (iOS) or Play Store (Android)
2. Open Expo Go
3. Scan the QR code shown in terminal

## 🛠️ Common Commands

```bash
# Start development server
npm start

# Clear cache and restart
npx expo start --clear

# Run on Android emulator
npm run android

# Run on iOS simulator (Mac only)
npm run ios

# Run on web browser
npm run web
```

## ⚠️ First Time Setup?

If this is your first React Native project:

1. Install Node.js from https://nodejs.org
2. Extract this folder
3. Open terminal in this folder
4. Run `npm install`
5. Run `npm start`
6. Scan QR code with Expo Go app

## 💡 Tips

- **Shake device** to open developer menu
- **Two-finger tap** to reload app
- **Use tunnel mode** if QR scan doesn't work: `npx expo start --tunnel`
